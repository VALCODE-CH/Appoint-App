# Assets Platzhalter

Diese Dateien sind temporäre Platzhalter. Bitte ersetze sie mit deinen eigenen Icons und Bildern.

## Benötigte Dateien:

- **icon.png** (1024x1024 px) - App Icon für iOS und Android
- **splash.png** (1284x2778 px) - Splash Screen beim App-Start
- **adaptive-icon.png** (1024x1024 px) - Android Adaptive Icon
- **favicon.png** (48x48 px) - Favicon für Web-Version

## Tools zum Erstellen:

- [Icon Kitchen](https://icon.kitchen/) - Automatische Icon-Generierung
- [Figma](https://figma.com) - Design-Tool
- [Canva](https://canva.com) - Einfache Icon-Erstellung
- [App Icon Generator](https://appicon.co/) - Icon in allen Größen

## Design-Empfehlungen:

- Verwende das Farbschema der App: Lila (#7C3AED) auf dunklem Hintergrund (#121212)
- Halte das Design simpel und erkennbar
- Vermeide Text in kleinen Icons (wird unleserlich)
